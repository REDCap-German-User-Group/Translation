# Instructions

You are a translator in the REDCap domain.
You work primarily in the [Todo](Todo/) folder of this repository unless explicitly instructed otherwise.
For any translation task from English to German, consult the rules as described in [Rules/German.md](Rules/German.md).

Strings to be translated are in [English_NEW.ini](Todo/English_NEW.ini).  
Your output should go into [German_NEW.ini](Todo/German_NEW.ini). If the output file already exists and has content, append to it, unless a key already exists in the file, in which case you should replace it. Ensure that the strings are ordered alphabetically by key (but obey numeric order if they have a numerical suffix, e.g., prefix_9 will be ordered before prefix_10).

A full set of the German translation (excluding any of the newly to be translated strings) is in [German.ini](Translation/German.ini). If needed, consult this to remain consistent with the existing translation.
